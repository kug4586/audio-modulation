주제 : 음성변조란 무엇인가

계기 : 통화 프로그램에 있는 음성변조 기능을 사용해보고, 그거에 흥미를 느껴서 음성변조 프로그램을 제작하고 싶어서

소리 : 공기나 물 같은 매질의 진동에 의해 생긴 파동을 귀가 감지한 것
음파 : 위에 밑줄 친 것
소리와 음파의 상관관계

소리의 3요소	음파
세기		진폭
높낮이		진동수
음색		파형

음성변조의 과정
녹음 >> 소음 제거 >> 노멀라이징 >> 주파수 변조

변조 : 데이터가 담긴 신호의 진폭이나 주파수를 변환해서 전송하는 통신기술.

펄스 부호 변조 (Pulse Code Modulation) :
 - 아날로그 -> 디지털
 - A.H.Reeves가 고안
 - 주로 녹음에 씀
 - 사용하기 쉽다.
 - 표본화 (sampling) :
	아날로그 신호에서 일정한 시간 간격마다 정보 추출,
	추출된 정보는 PAM(펄스 진폭 변조, Pulse Amplitude Modulation)이라 한다.
	표본화 정리에 따라, PAM의 주파수는 전송 신호의 주파수의 2배 이상으로 해야 원
	래신호를 재현할 수 있다.  
 - 양자화 (quantizaton) :
	신호의 원래 진폭을 이산적인 값으로 변환
	ex. 신호의 진폭이 최대가 10, 최소가 –10일 때,
	    진폭을 {x | [-10, 10] 사이에 있는 정수}로 설정
	표현할 수 있는 값은 비트 수에 따라 다름, 대부분 8bit를 씀
 - 부호화 (coding):
	양자화의 결괏값을 이진수로 변환

FFT (Fast Fourier Transform, 고속 푸리에 변환) :
푸리에 변환을 빠르게 수행하는 알고리즘
 - 푸리에 변환 : 주기함수 -> 삼각함수,
현실에서 일어나는 대부분의 현상은 시간에 대한 함수로 나타낼 수 있다. 그리고 푸리에 변환은 그 현상들을 주파수에 대한 함수로 바꿀 수 있게 해준다. 덕분에 전자전기공학에서 유용히 쓰인다.

노멀라이징 :
오디오 파일은 파일마다 음량이 다르다. 이때, 다 다른 음량을 균일하게 만들어 주는 것

python :
귀도 반 로섬이 창시한 프로그래밍 언어, 직관적으로 읽을 수 있어서 초심자들의 입문 언어,
데이터 처리에 뛰어나 인공지능 분야에서 자주 쓰임

사용한 모듈 :
*모듈 : 함수나 클레스를 불러올 수 있는 파일
*라이브러리 : 소프트웨어 개발에 쓸 수 있는 데이터 저장고 (window = 폴더)
 - pyaudio : python을 사용해 오디오 입출력을 할 수 있는 라이브러리
 - wave : 컴퓨터의 wav파일을 불러올 수 있는 라이브러리
 - librosa : 오디오 분석 라이브러리

실제 구현 
1. 녹음 : 구현됨
2. 노이즈 제거 : 구현 안 됨
3. 노멀라이즈 : 구현 안 됨
4. 주파수 변조 : 구현됨

노이즈와 노멀라이즈가 해결되지 않아 pitch, 즉 소리의 높낮이(주파수)를 바꾸면 노이즈까지 같이 변화돼서 노이즈가 많이 낀다.

결론 : 
컴퓨터에서 녹음하고 오디오를 이산적, 수치로 표현하는 것은 할 수 있었으나, 공부가 부족해 그 수치들을 직관적으로 알아볼 수가 없었다. 그 수치를 그래프나 스펙트럼으로 보여주려 했으나, 그 그림을 이해하기도 어려웠다. 때문에, 음성 데이터가 숫자로 표현됐음에도 불구하고 그 데이터를 다루지는 못했다.
이번 발표를 통해 부족한 점을 알았으니, 시간이 난다면 1인 1프나 교과 세특으로 푸리에 변환에 대해 탐구활동을 하거나 이 프로그램을 제대로 만들 생각이다.
